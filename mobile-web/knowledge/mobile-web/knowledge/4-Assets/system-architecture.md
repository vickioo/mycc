## 📚 记忆系统

### 三层结构

| 层级 | 文件 | 更新频率 | 用途 |
|------|------|----------|------|
| 短期 | `status.md` | 每日 | 当前状态快照 |
| 中期 | `context.md` | 每周 | 本周上下文 |
| 长期 | `about-me/` | 按需 | 用户画像 |

### Hooks 配置

```json
{
  "hooks": {
    "UserPromptSubmit": [
      {
        "command": "cat 0-System/status.md"
      }
    ]
  }
}
```

## 🚀 移动端架构

### V3 特性

- 聊天窗口 (类微信)
- 历史消息 (50 条)
- 状态反馈
- 快捷操作
- 移除导航

### API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 聊天界面 |
| `/api/chat` | POST | 发送消息 |
| `/api/history` | GET | 获取历史 |
| `/api/clear` | POST | 清空历史 |
