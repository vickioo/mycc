## 🎨 设计规范

### Apple 设计原则

1. **内容优先**
   - 减少装饰元素
   - 突出核心内容
   - 去除非必要 UI

2. **留白艺术**
   - padding: 12px-20px
   - margin: 8px-16px
   - gap: 10px-16px

3. **层次分明**
   - 阴影：`box-shadow: 0 2px 10px rgba(0,0,0,.1)`
   - 颜色：主色/次要/提示
   - 层级：Header > 卡片 > 背景

4. **流畅动画**
   - 过渡：`transition: all .3s`
   - 动画：`animation: fadeIn .3s ease`
   - 帧率：60fps

5. **无障碍**
   - 对比度：> 4.5:1
   - 点击区域：> 44x44px
   - 语义化 HTML
