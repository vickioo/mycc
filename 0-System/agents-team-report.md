## 📋 使用示例

### 提交后台任务
```python
from queue.task_queue import TaskQueue
q = TaskQueue()

# 提交任务后无需等待
q.submit({
    "name": "数据分析",
    "command": "python3 analyze.py",
    "target": "cc",
    "timeout": 3600
})
```

### 手机息屏场景
```
用户手机息屏
  ↓
Termux Worker 停止
  ↓
CC Worker 自动接管
  ↓
任务继续执行
  ↓
结果保存到队列
```
