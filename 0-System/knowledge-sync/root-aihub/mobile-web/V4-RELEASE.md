## 🚀 使用指南

### 开始对话
1. 访问 http://localhost:8772/chat
2. 选择 Agent (Qwen/Claude/Gemini)
3. 输入问题
4. 按 Enter 发送

### 快捷命令
1. 点击侧边栏命令
2. 自动填充到输入框
3. 按 Enter 执行

### 查看历史
1. 侧边栏查看最近对话
2. 点击历史项快速复用
3. 最多保存 50 条
