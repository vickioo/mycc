## 🌐 访问方式

### 方式 1: 本机浏览器

```
http://localhost:8765
```

### 方式 2: 手机浏览器 (同一 WiFi)

1. **查看电脑 IP** (在 U22 中):
   ```bash
   # 方法 1
   ip addr | grep "inet " | grep -v 127.0.0.1
   
   # 方法 2 (如果上面失败)
   cat /etc/hosts | grep localhost
   ```

2. **手机访问**:
   ```
   http://<电脑 IP>:8765
   ```

### 方式 3: 通过 gate 服务器 (远程访问)

```bash
# 在 gate 服务器上添加反向隧道
ssh -R 8765:localhost:8765 CC

# 然后通过 gate 服务器 IP 访问
http://<gate 服务器 IP>:8765
```

点击卡片切换助理：

在输入框中输入：
- **Shell 命令**: `ps aux | grep claude`
- **自然语言**: `查看当前任务`
- **服务管理**: `sv restart ssh-gate`

## 🔧 服务管理

### 启动服务

```bash
cd /root/air/qwq/mobile-web
python3 app.py
```

### 后台运行

```bash
nohup python3 /root/air/qwq/mobile-web/app.py > /tmp/mobile-web.log 2>&1 &
```

### 停止服务

```bash
pkill -f mobile-web
```

### 查看日志

```bash
tail -f /tmp/mobile-web.log
```

## ⚠️ 注意事项

### 网络要求

- 手机和电脑必须在**同一 WiFi 网络**
- 确保防火墙允许 8765 端口

### 安全说明

- ⚠️ **当前无密码验证**
- 🏠 **仅限内网使用**
- 🔒 不要开放到公网

### 性能

- 命令执行超时：30 秒
- 复杂命令可能较慢
- 建议用于简单查询和管理
