## 🔄 使用流程

### 日常工作
1. 访问统一主页 http://localhost:8772/
2. 查看系统状态
3. 点击"今日日志"创建当日记录
4. 选择需要的应用进入

### 查看文档
1. 主页点击"知识库"
2. 使用搜索或目录导航
3. 查看 Markdown 文档

### 项目管理
1. 主页点击"项目看板"
2. 查看 Mermaid 图表
3. 更新任务状态
