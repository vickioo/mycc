### U22 容器限制

**问题**: 旧服务占用端口 8772，无法杀掉

**原因**:
- U22 (Ubuntu 22.04 for Android) 容器限制
- `ps` 命令不可用
- `fuser` 命令不可用
- `kill` 命令无法定位进程

**影响**:
- 新代码无法加载
- 旧服务继续运行
- 自然语言对话功能无法生效
