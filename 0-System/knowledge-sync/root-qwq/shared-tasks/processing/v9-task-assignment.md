## 🔗 协作机制

### 任务同步

```bash
# 任务文件位置
/root/air/qwq/shared-tasks/processing/

# 任务格式
{
  "id": "task-xxx",
  "title": "任务标题",
  "assigned_to": "qwq|gemini|mycc",
  "status": "pending|in_progress|completed",
  "priority": "P0|P1|P2"
}
```

### 通信方式

1. **任务文件**: `shared-tasks/processing/*.json`
2. **状态更新**: 修改 `status` 字段
3. **完成通知**: 移动到 `completed/`

### 代码审查

- 使用 `task` 工具进行代码审查
- 重要变更需双方确认
- 文档同步更新
