## 📋 常见语音交互方案

根据经验，微信文章中的语音交互通常是以下方案之一：

### 方案 1: Azure Speech
- **语音识别**: Azure Speech-to-Text
- **语音合成**: Azure Text-to-Speech
- **费用**: 免费额度 + 按量付费
- **API Key**: 需要申请

### 方案 2: Whisper + Edge TTS
- **语音识别**: OpenAI Whisper (本地)
- **语音合成**: Edge TTS (免费)
- **费用**: 免费
- **API Key**: 不需要

### 方案 3: 百度/阿里语音
- **语音识别**: 百度语音 API
- **语音合成**: 百度语音 API
- **费用**: 免费额度 + 付费
- **API Key**: 需要申请

### 方案 4: Sherpa-ONNX
- **语音识别**: Sherpa-ONNX (本地)
- **语音合成**: Sherpa-ONNX (本地)
- **费用**: 完全免费
- **API Key**: 不需要
