**创建时间**: 2026-03-09

## 🎯 目标

实现**伪自主**的持续进化能力：
- 会话中断后无缝继续
- 每次对话自动唤醒记忆
- 反思沉淀经验
- 推进任务进度

---

## 🏗️ 架构设计

```
┌─────────────────────────────────────────────────────────┐
│  用户发起对话                                            │
└─────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│  auto-evolve.sh (自主唤醒)                               │
│  1. 读取 QWEN.md (用户记忆)                              │
│  2. 读取 progress-tracker.md (任务状态)                 │
│  3. 读取 6-Diaries (对话历史)                           │
│  4. 生成唤醒报告                                         │
└─────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│  对话中推进任务                                          │
│  - 心跳监控 (每 5 分钟)                                    │
│  - 反思学习 (每 25 分钟)                                   │
│  - 更新进度追踪器                                        │
└─────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────┐
│  对话结束                                                │
│  1. 更新 progress-tracker.md                            │
│  2. 更新 QWEN-EVOLVING-MEMORY.md                        │
│  3. 写入 6-Diaries 日记                                   │
└─────────────────────────────────────────────────────────┘
```

## 🔄 进化循环

### 对话开始时
```bash
# 自动运行
/root/air/qwq/scripts/auto-evolve.sh
```

**效果**:
- 读取所有记忆文件
- 生成唤醒报告
- 创建今日思考
- 更新进化记忆

### 对话进行中
```bash
# 心跳监控 (后台运行)
while true; do
    /root/air/qwq/scripts/mobile-web-heartbeat.sh
    sleep 300
done
```

**效果**:
- 每 5 分钟检查状态
- 每 25 分钟触发反思
- 自动同步任务到 mycc

### 对话结束时
```bash
# 手动运行 (或提示用户)
# 1. 更新进度追踪器
# 2. 更新进化记忆
# 3. 写入日记
```

**效果**:
- 持久化当前进展
- 沉淀学习经验
- 下次对话可继续
