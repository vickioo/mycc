```
GET  /api/health           # 健康检查
POST /api/chat             # AI 对话
POST /api/create-daily-log # 创建日志
GET  /manifest.json        # PWA 配置
```

## 📱 PWA 功能

### 添加到主屏幕
1. 访问 http://localhost:8772/
2. 浏览器菜单 → "添加到主屏幕"
3. 桌面会出现 AI Team 图标

### 快捷方式
- 📚 知识库
- 💬 AI 对话
- 📝 今日日志

### 离线支持
- 缓存主页
- 离线访问知识库
- 离线消息队列
